# 🗺️ География на купюрах России

> Интерактивная образовательная карта, показывающая географические объекты, изображённые на банкнотах Российской Федерации

[![GitHub repo](https://img.shields.io/badge/GitHub-Repository-blue?style=flat-square)](https://github.com/anastasiareent-sudo/banknotes)
[![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)](LICENSE)

---

## 📋 О проекте

Данный проект представляет собой интерактивный веб-ресурс, созданный в рамках итогового индивидуального проекта. Карта позволяет наглядно изучить достопримечательности российских регионов, которые изображены на банкнотах Банка России — от классических выпусков до новейших купюр.

### 🎯 Цель
Систематизировать знания о регионах России и повысить интерес к изучению географии и истории страны через привычный повседневный предмет — банкноту.

### ✨ Возможности
- 🗂️ Просмотр всех номиналов российских рублей с привязкой к карте
- 📍 Точные географические координаты объектов на купюрах
- 🖼️ Сравнение изображения на банкноте и фотографии реального объекта
- 📚 Краткие историко-географические справки по каждому объекту
- 📱 Адаптивный дизайн для удобного просмотра на любых устройствах

## ⚙️ Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/anastasiareent-sudo/banknotes.git
cd banknotes
```

### 2. Настройка API-ключа Яндекс.Карт

> ⚠️ **Важно!** Для корректной работы карты необходимо получить API-ключ в [Кабинете разработчика Яндекс](https://developer.tech.yandex.ru/) и заменить заглушку в коде.

В файле `index.html` (или соответствующем файле подключения карты) найдите строку:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=****&lang=ru_RU"></script>
```

Замените `****` на ваш реальный API-ключ:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ_ЗДЕСЬ&lang=ru_RU"></script>
```

### 3. Запуск проекта
Просто откройте файл `index.html` в браузере или используйте локальный сервер:

```bash
# Например, с помощью Python
python -m http.server 8000
```

Затем перейдите по адресу: `http://localhost:8000`

| 100 ₽ | Москва | Спасская башня Кремля, Парк «Зарядье», Останкинская башня, Здание МГУ, Ржевский мемориал |

## 🧪 Тестирование и эффективность

В рамках проекта проведено анкетирование 25 респондентов:

- ✅ **96%** отметили рост интереса к изучению купюр после знакомства с картой
- ✅ **90%** узнали о новых дизайнах банкнот (в т.ч. о выпуске 1000 ₽ 2025 г.)
- ✅ **88%** подтвердили, что визуальная привязка помогает запоминать регионы
- ✅ Время поиска информации сократилось с **12–15 мин** до **1–2 мин**
